# 学习共享平台 - 用户使用指南

欢迎使用学习共享平台！这是一个支持题目、笔记上传分享的在线学习社区。

---

## 🎯 主要功能

### 📚 题库系统
- **搜索题目**：关键词搜索题目内容
- **浏览题库**：按标签分类浏览题目
- **上传题目**：支持文本、图片、PDF/Word 文档上传
- **收藏管理**：收藏喜欢的题目，方便复习
- **评论互动**：参与题目讨论，分享见解

### 📝 笔记系统
- **上传笔记**：分享学习笔记和资料
- **浏览笔记**：发现优质学习笔记
- **收藏管理**：整理和收藏学习资源

### 🎓 班级协作
- **创建班级**：组建学习小组
- **加入班级**：参与班级学习活动
- **成员管理**：管理班级成员

### 🤖 AI 助手
- **智能问答**：AI 助手实时回答学习问题
- **图片对话**：上传图片进行问答
- **会话管理**：多会话历史记录

### 🎨 个性化设置
- **主题切换**：6 种精选配色方案
- **自定义背景**：上传个人背景图片
- **个人主页**：展示个人信息和作品

## 🚀 快速上手

### 1. 注册/登录
1. 访问网站首页
2. 点击右上角"登录 / 注册"
3. 输入邮箱和密码完成注册
4. 使用邮箱和密码登录

### 2. 浏览题库
1. 点击导航栏"题库"
2. 输入关键词搜索
3. 点击题目查看详情
4. 使用标签筛选题目类型

### 3. 上传题目
1. 登录后点击右上角用户名 → "上传题目"
2. 选择上传方式：
   - **文本上传**：直接输入题目内容
   - **图片上传**：上传题目图片
   - **文档上传**：支持 PDF、Word 文档
3. 填写题目信息（题目标题、内容、答案等）
4. 添加标签分类
5. 点击"发布"完成上传

### 4. 使用 AI 助手
1. 点击右下角浮动 AI 按钮
2. 输入问题，支持文字和图片
3. AI 助手实时回答
4. 可创建新对话或查看历史

### 5. 个性化设置
1. 点击导航栏调色板图标
2. 选择喜欢的主题配色
3. 点击"上传背景图片"自定义背景
4. 设置自动保存，刷新后保持
